Step 1 — Select Device Type

USB / Serial

Connect using a USB cable. Supports ESP32 boards with USB-Serial.

Web Serial not supported in this browser.
Bluetooth

Connect wirelessly via BLE. The device advertises with a device-derived setup name in provisioning mode. If prompted, use the device-derived 6-digit passkey shown by the device setup output.

Web Bluetooth not supported in this browser.
WiFi AP

Device broadcasts <%= setupApLabel %> WiFi. Connect your computer to that network, then click below.

Works in all browsers
WiFi AP steps:
  1. Power on your ESP32 (first boot / factory reset).
  2. On your computer, open WiFi settings and connect to <%= setupApLabel %>. The current firmware starts this setup AP open unless you later change the policy.
  3. Come back here and click Connect via WiFi. This server will reach the device at <%= setupApIp %> and send the same config the portal would apply.
USB / Bluetooth require Chrome or Edge (v89+). WiFi AP works in any browser — no cable needed.
